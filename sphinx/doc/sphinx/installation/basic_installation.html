Basic Installation¶

Note

The following instructions will help you to install a packaged version of openage on your computer. If you have problems with that please take a look into Package Instructions.

Linux Installation¶

We provide packages for different Linux distributions. Please take a look for distribution packages at Repology or Bintray for any future updates on *.deb and AppImage packages.

Windows Installation¶

We provide both installer and portable packages for Windows 32-bit and 64-bit versions. The latest release installer you find at Github. Our nightly builds you will find over here.

macOS Installation¶

For MacOSX we don’t have packages, but it should build fine. Here you can check the build status, while the instructions for Dependency installation should help you with finding out what you need to get openage building.
